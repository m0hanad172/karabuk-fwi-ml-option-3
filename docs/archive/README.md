# Archive — historical project notes

These documents record how the project evolved through its
development phases. They are preserved verbatim for academic context
and audit traceability. **They are not the operational reference** —
for current run instructions, architecture, schema, or deployment
plans, see the active documents in the parent [`docs/`](../) folder.
